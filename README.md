Employee Task Dashboard

This is a simple task management web app built using React.
It helps users to manage daily tasks easily by adding, editing, deleting, and tracking tasks.

Live Project

You can view the live project here:
https://employee-task-dashboard-liard.vercel.app/


 Features 

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Search tasks by title
- Filter tasks (All, Completed, Pending, High Priority)
- Shows task details like priority, status, and due date
- Dark mode and light mode support
- Fully responsive (works on mobile and laptop)

Technologies Used:

- React.js
- JavaScript
- HTML
- CSS
- Vite
- LocalStorage (for saving tasks)


 How to Run Project Locally

1. Download or clone this repository
2. Open in VS Code
3. Install dependencies: npm install
